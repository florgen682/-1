# Лабораторная работа: Разработка 3D-симулятора автомобильного крана

Проект представляет собой учебный трехмерный симулятор управления автомобильным краном в условиях портового терминала. Программа написана на языке Python с использованием библиотеки Ursina Engine.

## Порядок запуска приложения

Для запуска симулятора на локальном компьютере необходимо выполнить следующие шаги:

1. Убедиться, что в системе установлен Python версии 3.9 или выше.
2. Установить графический движок Ursina через командную строку:
   ```bash
   pip install ursina
   ```
3. Запустить скрипт с исходным кодом:
   ```bash
   python main.py
   ```

## Описание технической реализации

В ходе работы над проектом были решены следующие задачи:
* Организована правильная деревовидная иерархия объектов через параметр `parent`. Это позволило жестко связать кабину, колеса и платформу с шасси грузовика, чтобы модель не распадалась при перемещении.
* Реализовано векторное движение машины по направлению её локальной оси `forward`, что имитирует базовую логику автомобильного разворота вместо простого сдвига по осям координат.
* Написан алгоритм расчета динамического троса. С помощью метода `to_global()` программа точно определяет координаты конца стрелы при любом её наклоне или повороте башни, после чего выравнивает и масштабирует модель троса.
